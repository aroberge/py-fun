Blank windows

In this tutorial we show how to create blank windows in both Python, using pyglet, and javascript.

Blank windows with Python and pyglet

Using Python and pyglet, it is very easy to create a window: only three lines of code are needed.

import pyglet
game_window = pyglet.window.Window(caption="My game")
pyglet.app.run()

Go ahead, and try it!

Blanking the window

You may have noticed that the window that was created had some weird content in it. To truly blank the window, we need to call the clear() method already defined for us in pyglet. 

import pyglet

game_window = pyglet.window.Window(width=800, height=600, caption="My game")

@game_window.event
def on_draw():
    game_window.clear()

pyglet.app.run()

Note the use of a decorator so that the on_draw() function will be called whenever an relevant window event occurs

That's it! Try it yourself!
